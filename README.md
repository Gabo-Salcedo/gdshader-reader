# GDShader Reader
A specialized Obsidian plugin for viewing Godot Engine `.gdshader` files with advanced GLSL syntax highlighting and integrated visual references.

## Installation

### From Obsidian
1. Open Settings ‚Üí Community Plugins
2. Search "GDShader Reader"
3. Install and Enable

### Manual
1. Download latest release
2. Extract to `<vault>/.obsidian/plugins/gdshader-reader/`
3. Reload Obsidian and enable in settings

## Usage

1. **Open a `.gdshader` file** ‚Üí Automatic syntax highlighting
2. **Add reference** ‚Üí Click upload button üì∑ or drag & drop image/video
3. **Customize theme** ‚Üí Use palette button üé® in toolbar
4. **Control videos** ‚Üí Use play/pause, loop, audio, and speed buttons

## Configuration

Access via `Settings ‚Üí GDShader Reader`:
- Choose storage location (global/relative)
- Select color preset or customize individual colors
- Adjust image panel background
